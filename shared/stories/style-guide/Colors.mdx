# Colors

Looppit의 컬러 시스템은 **다크 배경**과 **보라색 포인트**를 중심으로 구성되어 있습니다.
카드 기반 레이어링 시스템을 통해 콘텐츠의 계층 구조를 명확하게 표현하며,
투명도를 활용한 세밀한 색상 변형으로 다양한 UI 상태를 표현합니다.

아래에 표시된 색상들은 모두 **Tailwind CSS className**으로 사용됩니다.
